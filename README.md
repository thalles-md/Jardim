# SAM 
## Site Pessoal

Olá! Eu criei esse lugar para poder criar e me expressar na internet de uma forma que eu tenha um pouco mais de controle. A seção Escritas tem meus queridos textos (eu amo escrever) e na seção Postagens tem postagens periódicas que eu irei fazendo, é um lugar mais informal e descontraído que nos meus textos.

Estruturei esse site (a parte de código e funcionalidades) inteiramente usando IA, apesar de, devo admitir, não apoiar esse tipo de prática. 
De toda forma, todo o conteúdo (escritas, postagens) é inteiramente escrito/produzido por mim.
